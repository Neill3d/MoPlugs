
 == MoPlugs ==

 Скрипты и подключаемые модули для Autodesk MotionBuilder.

Поддерживаемые версии
- 2014, 2015, 2016, 2017, 2018, 2019

Плагины
* camera linkvis
* constraint - aim sprite
* import geometry cache
* mo_graphics - MoPlugs Renderer
* constraint - column
* renderer colors
* shader - gpu particles
* shader - skybox
* tool bake projections
* tool blend shape (and sculpt brush)
* tool render layers (and command line renderer)
